# Product Requirements â€“ Clueless AI Closet MVP

## One-Line Description
Helps people who struggle with styling visualize outfits on a person before wearing them.

## Primary User Story
As a user, I want to see how my selected clothes look together on a model so I can confidently choose outfits.

---

## MUST-HAVE MVP FEATURES

1. Upload clothing images from device gallery
2. Automatic background removal on upload
3. Digital closet (grid view of isolated clothing items)
4. Outfit canvas to select and combine items
5. AI outfit visualization using **Gemini NanoBanana 2.5**
   - Prompt-based generation
   - Model/person wearing selected outfit
6. Save generated visualization
7. Download visualization image
8. Delete items from digital closet
9. Authentication (login / signup)

---

## NICE-TO-HAVE (Only if time remains)
- Regenerate outfit variations
- Simple item tags (top / bottom / shoes)

---

## NOT IN MVP (Strictly Excluded)
- AI outfit recommendations
- Calendar outfit planning
- Social feeds
- Mobile app
- Advanced try-on realism
- User analytics / dashboards
- Custom UI polish or animations

---

## Success Metrics
- End-to-end flow works without breaking
- Outfit visualization generates successfully
- App is live and usable on the web
- MVP completed within 3 days

---

## Design Vibe
Minimal, clean, fast, functional

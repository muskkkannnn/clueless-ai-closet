## **4. AI TOOLING RESEARCH (FREE TOOLS)**

### **Recommended Free AI Stack**

#### **A. Background Removal**

**Primary Choice: BRIA RMBG-1.4**
- **URL:** https://huggingface.co/briaai/RMBG-1.4
- **License:** Creative Commons (non-commercial)
- **Why:** State-of-the-art background removal model trained on general stock images, e-commerce, and gaming content
- **Alternative:** RMBG-2.0 (improved version)
- **API:** Free via Hugging Face Inference API

```javascript
// Example usage
import { pipeline } from '@huggingface/transformers';
const pipe = pipeline("image-segmentation", "briaai/RMBG-1.4");
const result = pipe(imageFile);
```

#### **B. Image-to-Text (Clothing Description)**

**Primary Choice: Salesforce BLIP**
- **URL:** https://huggingface.co/Salesforce/blip-image-captioning-base
- **License:** Open source
- **Why:** Vision-Language Pre-training framework designed for both understanding and generation tasks with state-of-the-art results on image captioning
- **Sizes:** Base (faster) or Large (more detailed)

```javascript
// Example prompt engineering
const processor = BlipProcessor.from_pretrained("Salesforce/blip-image-captioning-base");
const prompt = "a photo of clothing items: "; // Conditional captioning
```

**Why Image‚ÜíText‚ÜíText-to-Image is Simpler:**
- Direct image-to-image (virtual try-on) requires specialized models like ControlNet
- Text intermediate step allows prompt engineering and control
- Stable Diffusion excels at text-to-image generation
- Trade-off: Less accurate garment details, but much faster to implement

#### **C. Text-to-Image (Outfit Visualization)**

**Primary Choice: Stable Diffusion 1.5**
- **URL:** https://huggingface.co/stable-diffusion-v1-5/stable-diffusion-v1-5
- **License:** CreativeML OpenRAIL M (permissive commercial use with safety restrictions)
- **Why:** Lightweight at 860M UNet, capable of generating photorealistic images from text prompts on consumer GPUs
- **API:** Free via Hugging Face Inference API

**Alternative: Stable Diffusion XL (better quality, slower)**

### **Complete Free AI Pipeline**

**Total Cost: $0** (within free tiers)

1. **User uploads clothing** ‚Üí Store in Supabase Storage
2. **Remove background** ‚Üí BRIA RMBG-1.4 API call
3. **Generate descriptions** ‚Üí BLIP API (3 calls for 3 items)
4. **Build prompt** ‚Üí Combine descriptions
5. **Generate visualization** ‚Üí Stable Diffusion API
6. **Save result** ‚Üí Supabase Storage

**Example Prompt Template:**
```
"A fashion model wearing {top_description}, {bottom_description}, and {shoes_description}. Full body photo, professional fashion photography, studio lighting, white background."
```

### **Limitations & Trade-offs**

‚ö†Ô∏è **Accuracy Issues:**
- BLIP generates captions without guarantees of exact color or pattern replication
- Stable Diffusion may interpret descriptions creatively
- Not a true "try-on" - visualizing similar items

‚ö†Ô∏è **Free Tier Constraints:**
- Hugging Face Inference API: Rate limited (1-2 req/second)
- Processing time: 3-10 seconds per visualization
- No guaranteed uptime (best effort)

‚ö†Ô∏è **Quality Trade-offs:**
- Garment accuracy: 60-75% (colors/patterns approximate)
- Model consistency: May vary between generations
- Perfect fit simulation: Not possible with this approach
